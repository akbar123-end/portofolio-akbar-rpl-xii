HTML CSS

1. HTML (HyperText Markup Language) HTML adalah bahasa markup yang digunakan untuk membuat struktur halaman web. HTML menggunakan tag untuk menampilkan berbagai elemen seperti teks, gambar, video, tabel, dan lainnya. 2. CSS (Cascading Style Sheets) CSS digunakan untuk mengatur tampilan dan desain halaman web, seperti warna, font, dan tata letak elemen.

Bootstrap 5

Bootstrap adalah framework CSS yang memudahkan pembuatan desain web yang responsif dan modern. Dikembangkan oleh Twitter, Bootstrap menyediakan berbagai komponen siap pakai, seperti grid system, tombol, form, navbar, dan lainnya, tanpa harus menulis kode CSS dari nol. 1. Kelebihan Bootstrap ✅ Mudah Digunakan – Hanya perlu menambahkan class pada elemen HTML. ✅ Responsif – Menggunakan grid system untuk tampilan yang fleksibel di berbagai perangkat. ✅ Kompatibel dengan Semua Browser – Didukung oleh hampir semua browser modern. ✅ Banyak Komponen Siap Pakai – Seperti navbar, modal, card, carousel, dll.

Javascript

JavaScript adalah bahasa pemrograman yang digunakan terutama untuk pengembangan web. Ini adalah bahasa yang berjalan di sisi klien (client-side) dan memungkinkan interaksi dinamis pada halaman web. JavaScript sering digunakan untuk membuat elemen interaktif seperti animasi, validasi formulir, efek transisi, dan komunikasi dengan server tanpa harus memuat ulang halaman (AJAX).

Laravel 6

Laravel adalah framework PHP yang digunakan untuk membangun aplikasi web dengan arsitektur MVC (Model-View-Controller). Laravel dikembangkan oleh Taylor Otwell dan dikenal karena sintaksnya yang elegan, kemudahan penggunaan, dan fitur bawaan yang lengkap untuk pengembangan web modern..

PHP & MySQL

PHP adalah bahasa pemrograman server-side yang digunakan untuk membuat aplikasi web dinamis, sedangkan MySQL adalah sistem manajemen basis data relasional (RDBMS) yang digunakan untuk menyimpan data. Kombinasi PHP & MySQL banyak digunakan dalam pengembangan website, seperti sistem manajemen konten (CMS), e-commerce, dan aplikasi berbasis data lainnya. .
